# 📚 Exam Scheduler

A simple and interactive **Exam Schedule Tracker** built using **HTML, CSS, and JavaScript** that helps students manage upcoming exams efficiently. Users can add exams, track countdowns, and visualize urgency with color-coded cards.

## 🚀 Live Demo

🔗 **View Demo:** https://venkateswarlu-j.github.io/ExamSchedules/

---

## ✨ Features

- Add exams with **name and date**
- View all upcoming exams in a dashboard
- Automatic **countdown calculation**
- Dynamic **color-coded urgency system**
  - 🔴 Red → Very close exams
  - 🟡 Yellow → Upcoming soon
  - 🟢 Green → Plenty of time left
- Delete exams anytime
- Data stored using **localStorage**
- Auto-updates countdown every page refresh
- Exams automatically sorted from nearest to farthest

---

## 📌 Application Flow

```text
User opens app
      ↓
Sees existing exams (loaded from localStorage)
      ↓
Clicks "Add Exam"
      ↓
Fills exam name + date
      ↓
Saves exam
      ↓
Dashboard displays exam cards with countdown
      ↓
Cards automatically color-code based on urgency
      ↓
User can delete exams anytime
      ↓
Every page load updates countdown automatically
```
